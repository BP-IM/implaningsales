Экспорт прошлой недели

Загрузите Excel файл с почасовой выручкой. Система автоматически рассчитает факт по дням и часам.

Формат файла

Файл должен быть в формате “Почасовая выручка” с колонками: Учетный день, Час закрытия, Количество гостей, Сумма со скидкой.
